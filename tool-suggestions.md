# TTRPG Tool Suggestions for GM Buddy

Ideas for future tools to assist Game Masters running campaigns in the Nimble system.

## Combat Management

- **Initiative Tracker**: A sidebar view that tracks turn order. Insert a code block with creature names and initiative bonuses, roll initiative for all at once, and display a sorted turn list with a "next turn" button.
- **Condition Tracker**: Tag creatures with status conditions (stunned, prone, poisoned, etc.) that auto-expire after a set number of rounds. Integrates with the initiative tracker.
- **Damage Calculator**: A quick-access modal for applying damage or healing to multiple creatures at once, factoring in resistances and vulnerabilities.

## Encounter Design

- **Encounter Builder**: Given a party size and level, suggest balanced encounter compositions by pulling from a creature database stored in notes.
- **Random Encounter Generator**: Define encounter tables with weighted probabilities per environment type. Roll to generate a random encounter with all relevant stat blocks linked.
- **Loot Generator**: Roll on configurable loot tables to generate treasure drops. Supports tiered tables (e.g., minor, moderate, major) and custom item pools.

## World Building

- **NPC Generator**: Randomly generate NPC names, personality traits, motivations, and appearance from user-defined tables. Insert the result as a formatted note or callout.
- **Settlement Generator**: Create quick settlement profiles with population, notable locations, factions, and rumors from configurable random tables.
- **Calendar and Timeline Tracker**: Define a custom in-game calendar. Track the current date, log events, and calculate durations between events. Display a timeline view in notes.

## Session Management

- **Session Log Template**: A command that inserts a pre-formatted session log template with date, attendees, recap, key events, and follow-up sections.
- **Player Tracker**: A sidebar view summarizing each player character's key stats, current HP, and active conditions. Updated manually or via quick commands.
- **Quest Board**: Track active, completed, and failed quests. Each quest is a note linked from a central board view. Mark quests as progressing, completed, or failed from the board.

## Rules Reference

- **Quick Rule Lookup**: Index key rules from the Nimble system into a searchable database. A command opens a fuzzy search modal to find and insert rule references or summaries.
- **Ability Card Renderer**: Define abilities, spells, or features in a structured code block format and render them as styled cards in reading view.

## Randomization Utilities

- **Random Table Roller**: Define weighted random tables in notes (using a code block format) and render them with a "Roll" button. Supports nested table references (roll on one table, then roll on a sub-table based on the result).
- **Name Generator**: Generate random names from phonetic rules or predefined name lists, filtered by culture or species.
