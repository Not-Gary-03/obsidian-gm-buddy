# GM Buddy

Handful of tools and utilities for Game Masters running tabletop RPG campaigns in [Obsidian](https://obsidian.md).

## Status

This plugin is in early development. Features are being actively planned and built.

## Development

### Prerequisites

- Node.js v16+
- npm

### Setup

```bash
npm install
```

### Build

```bash
# Watch mode (development)
npm run dev

# Production build
npm run build
```

### Manual installation

Copy `main.js` and `manifest.json` to your vault at:

```
<vault>/.obsidian/plugins/obsidian-gm-buddy/
```

Then reload Obsidian and enable the plugin in **Settings → Community plugins**.
